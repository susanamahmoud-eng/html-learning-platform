# 🎮 HTML Interactive Learning Platform

A gamified HTML learning platform where you can learn, practice, and earn points while mastering HTML tags!

## 🚀 Live Demo

**[Play Now →](https://susanamahmoud-eng.github.io/html-learning-platform/)**

## ✨ Features

- **🏆 Progress Tracking** - Earn points and unlock badges as you learn
- **📘 Interactive Lessons** - Learn HTML tags (h1, p, img, a) with live examples
- **✍️ Code Practice** - Write HTML code and see the live preview instantly
- **🧠 Quiz System** - Test your knowledge with interactive questions
- **🎯 Gamification** - Points system with 3 achievement levels:
  - ⭐ Beginner (20+ points)
  - 🔥 Intermediate (50+ points)
  - 🏆 Pro (100+ points)

## 📊 Point System

| Activity | Points |
|----------|--------|
| Learn a tag | +5 |
| Run code | +10 |
| Correct quiz answer | +15 |

## 🎯 How to Use

1. **Learn** - Click on HTML tag buttons to see live examples
2. **Practice** - Write your own HTML code in the editor and click "Run Code"
3. **Quiz** - Answer questions to test your knowledge
4. **Track Progress** - Watch your points grow and unlock badges!

## 🛠️ Technologies

- **HTML5** - Structure
- **CSS3** - Styling & animations
- **Vanilla JavaScript** - Interactivity

## 📝 Current Tags Covered

- `<h1>` - Headings
- `<p>` - Paragraphs
- `<img>` - Images
- `<a>` - Links

## 🚀 Getting Started Locally

1. Clone the repository:
```bash
git clone https://github.com/susanamahmoud-eng/html-learning-platform.git
```

2. Open `index.html` in your browser

That's it! No dependencies or build tools needed.
